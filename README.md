## Flask + Socket.io

### Set up

#### Step 1

```js
py -3 -m venv venv
```

#### Step 2

```js
pip3 install -U pip virtualenv
```

#### Step 3

```js
virtualenv --system-site-packages -p python ./venv
```

#### Step 4

```js
pip install Flask
```

### Install flask-socketio

```js
pip install flask-socketio
```

### Auto reload

```js
flask --app app.py --debug run
```

### Run using jwt not error

```js
python app.py
```
